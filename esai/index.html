Materi Pembelajaran SIGWeb

Praktikum SIG berbasis Web mempelajari cara untuk memvisualisasikan peta dalam bentuk peta interaktif. Peta interaktif dalam format digital ini, tidak hanya merupakan wujud peta digital. Dalam peta yang divisualisasikan menggunakan konsep SIG Web, pengguna peta dapat mengakses peta secara interaktif, seperti memperbesar layer peta, mengaktifkan dan menonaktifkan layer, dan sebagainya.

    Materi praktikum SIG berbasis web sebelum UTS, terdiri dari:
  1. Dasar-Dasar HTML: HTML pada pembuatan peta interaktif digunakan untuk mendifinisikan isi dari konten yang akan ditampilkan.
  2. Dasar-Dasar CSS: CSS merupakan cara untuk mengatur style pada tampilan konten yang sebelumnya telah dibuat menggunakan HTML.
  3. Dasar-Dasar JavaScript: Javascript dimanfaatkan untuk mengatur interaksi yang dapat dilakukan oleh pengunjung website.
  4. Pengaplikasian Bootstrap: Bootstrap merupakan metode yang dapat dimanfaatkan untuk membuat laman web. Melalui bootstrap, pembuatan laman web tidak perlu dimulai dari awal (scratch).
  5. Pengaplikasian LeafletJS: Leaflet JS merupakan bagian dari Javascript yang familier digunakan untuk visualisasi peta. Pada Leaflet JS, data geospasial, seperti data basemap, data vektor dengan tipe geojson, dan sebagainya dapat ditampilkan pada laman web.
